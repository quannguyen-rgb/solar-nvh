📦 Sao Lưu & Khôi Phục Bảng Giá
App chạy offline. Nhập dữ liệu bằng 2 cách:
• 📊 Nhập từ Excel → chọn file NVH_Solar_Database_Template.xlsx (cần internet lần đầu để tải thư viện)
• 📥 Khôi phục JSON → khôi phục từ file .json đã xuất trước đó
Sau khi nhập, nhấn 💾 Lưu DB để lưu vào máy.
